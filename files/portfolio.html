The honest version of my story

I grew up in Mumbai obsessed with one question: why do some cities work and others don't? That curiosity eventually pulled me into AI — not because I wanted to build chatbots, but because I realised machine learning is one of the few tools that can actually answer questions at city scale.

My BTech in AI & ML at DJSCE gave me the foundations. The projects I built there — RAG pipelines for clinical nutrition, ingredient recognition systems, published research — gave me the confidence to push further. Now I'm at UIC Chicago, the city itself becoming a kind of living dataset.

I care about the full stack: from fine-tuning a model on 40K examples to deploying it behind a FastAPI service with CI/CD. I'm not interested in demos that never ship.

Natural Language → Bash via LLM Fine-Tuning

Geospatial Analysis of Urban Food Deserts
↗ Live Demo ↗ GitHub

03
AI-Driven ADHD Diet Recommendation System
"ADHD is a spectrum. Nutrition guidance shouldn't be one-size-fits-all — it should know if it's talking to a 7-year-old or a 45-year-old."

End-to-end clinical AI using DeepSeek-R1 and LLaMA-3 Vision with a dual RAG pipeline — one path for peer-reviewed dietary research, one for patient context. Built on Kubeflow with batched GPU inference that cut latency by 22%. Validated across four age groups. This project was about clinical trust as much as raw accuracy.
